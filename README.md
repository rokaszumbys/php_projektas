# Slaptažodžių generatorius ir saugykla

Tai paprastas PHP OOP projektas, skirtas vartotojų registracijai, prisijungimui, slaptažodžių generavimui ir saugojimui duomenų bazėje.

## Funkcijos

- Vartotojo registracija
- Prisijungimas
- Slaptažodžio generavimas pagal pasirinktus parametrus
- Slaptažodžio saugojimas duomenų bazėje
- Slaptažodžių užkodavimas
- Paprasta vartotojo sąsaja

## Naudotos technologijos

- PHP
- MySQL
- PDO
- HTML
- GitHub
